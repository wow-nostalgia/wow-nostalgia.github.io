# Інструкції для Claude Code — цей проєкт

## Перевірка в браузері (Playwright)

Playwright-перевірку (install playwright/http-server, запуск сервера,
скрін/перевірка в браузері) запускати **тільки перед комітом**, і
**тільки після явного підтвердження користувача** — не автоматично
після кожної правки чи навіть автоматично перед кожним комітом без
запиту.
