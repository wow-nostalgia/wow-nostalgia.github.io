Аналітика

  • Графіки будуються на основі тих самих даних, що й Рейтинг DPS (uwu-logs.xyz, режим ЦЛК 25 ХМ).

Завантаження даних...

Топ-50 / Топ-100 / Топ-200 серверу по спеку

Кількість гравців гільдії в топ-50, топ-100 та топ-200 серверу для кожної спеціалізації.

Популярність спеків

Скільки активних логів (гравець + спек) є по кожній спеціалізації.

Слабка ланка по боссах

Середній DPS гільдії по кожному боссу ЦЛК (боси без чіткого DPS-чеку — Валітрія, Анубарак, Toravon — не враховані).

Середній DPS рейду по боссу в часі

Середній DPS гільдії (тільки дамаг-спеки) по обраному боссу ЦЛК, по датах рейдів.

Сумарний DPS рейду по боссу в часі

Сума DPS усіх гравців рейду (дамаг-спеки, танки і хіли) по обраному боссу ЦЛК, по датах рейдів.

Середній Score по спеку

Середній показник Score серед гравців гільдії на спеку.

Розподіл ролей

За активними логами спеків. Feral Combat зараховано як DPS — дані не розрізняють кет/друїд-танк.

Гільдія проти серверу

Скільки гравців гільдії потрапляють в топ-50/100/200/500 серверу (за найкращим спеком кожного гравця).

Елітні виконавці

Кількість гравців гільдії з найкращим Score не нижче порогу.

Гільдія проти легіонерів

Завантаження...

Середній результат по класу

Який клас в середньому показує кращий результат (Score) в рейдах.

Скільки гравців показують який результат

Кожен стовпчик — діапазон Score. Висота стовпчика — скільки гравців гільдії в нього потрапили.

Топ-3 ДД кожної спеціалізації

3 найкращих гравця кожної дамаг-спеки за результатом (Score). Наприклад, Шаман Елементаль і Шаман Підсилення рахуються окремо, а не разом.

Топ-3 хілери кожної спеціалізації

3 найкращих гравця кожної хіл-спеки за швидкістю хілу (HPS) на Валітрії Дрімвокер — це єдиний бос ЦЛК, де результат залежить саме від хілу, а не від дамагу. Жрець Дисципліна і Жрець Свято рахуються окремо.

Відвідуваність рейдів

Топ-15 гравців за кількістю рейдів, в яких вони брали участь (за даними логів потів).

Чи допомагають поти показати кращий результат?

Кожна крапка — один гравець. Праворуч — хто п'є більше потів за рейд. Вище — у кого кращий результат (Score). Якщо крапки тягнуться знизу-зліва до верху-справа — поти допомагають.
